# Temper

Precision tools for audio professionals, by Temper.

Sound design is creative work.
But most of the time gets spent on the repetitive: swapping samples, managing properties, preparing libraries, placing markers. The creative decisions that define your sound get squeezed into whatever time is left.
Temper exists to reverse that. Every tool we build removes a layer of tedious work so you can operate at scale without sacrificing the artistry.
Faster output. Better decisions. More time to be creative.
Many tools. One workflow.

See demos, documentation, and purchase options at
[tempertools.com](https://tempertools.com).

## What's inside

- **Temper Vortex**: Randomize and roll item properties across layered
  tracks, with inheritance and variation slots.
- **Temper Vortex Mini**: The Vortex randomization engine, focused on a
  single item and its layers.
- **Temper Imprint**: Capture and paste routing, FX chains, envelopes,
  and sends between tracks.
- **Temper Mark**: Transfer take markers onto the rendered WAV file.
- **Temper Alloy**: Combine WAV variants into a single multichannel file.
- **Temper Slice**: Bounce items to structured WAV files with embedded
  metadata.
- **Temper Slice Mini**: Convert stereo WAV files to mono.
- **Temper Archive**: Package a full REAPER project folder for handoff
  or cold storage, cross-platform.

## Install

The scripts ship as a ReaPack repository.

1. Install [ReaPack](https://reapack.com) inside REAPER if you haven't
   already.
2. Open **Extensions > ReaPack > Import repositories**.
3. Paste this URL:

   ```
   https://raw.githubusercontent.com/tempertools/temper-tools/main/reapack/index.xml
   ```

4. Click **OK**, then open **Extensions > ReaPack > Browse packages** to
   install individual tools.

Trial and licensing details are handled inside each script on first
launch. Full purchase options live at
[tempertools.com](https://tempertools.com).

## Requirements

- [REAPER](https://www.reaper.fm)
- [ReaPack](https://reapack.com)
- [ReaImGui](https://github.com/cfillion/reaimgui), installable via
  ReaPack. Every Temper script uses ReaImGui for its interface.
